## 使用tensorflow对目标图像进行分类

### 系统需求：
#### Windows：
- Python 3.5.*
- OpenCV 3.2.*
- Tensorflow 1.0.1

#### Linux:
##### 推荐使用Ubuntu 14.04/16.04 LTS
- Python 2.7.*
- OpenCV 2.4.*
- Tensorflow 1.0.1

### 使用方法：
- 在根目录下打开终端(cmd, powershell or terminal)
- 在终端中输入
`
python test.py [<video source>]
`
- 拍照之后将图像分割成上下两部分分别识别
- 识别结果存放在`images_classify/results/results.json`
